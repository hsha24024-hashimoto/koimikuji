恋みくじ（最強版）
シンプルな操作で今日の恋愛運を占うことができるWebアプリケーションです。

概要
このアプリは、HTML、CSS、JavaScriptを使用して作成されたフロントエンド・アプリケーションです。ボタンをクリックすることで、あらかじめ用意された運勢（大吉、中吉、小吉、凶）の中からランダムに一つが表示されます。

機能
1.ボタンをクリックするとランダムに運勢を表示

2.ピンクを基調とした可愛らしいデザイン

3.シンプルで直感的なユーザーインターフェース


使い方
1.すべてのファイルを同じフォルダに配置します。

2.ブラウザで index.html を開きます。

3.「おみくじを引く」ボタンをクリックして、運勢を確認します。
